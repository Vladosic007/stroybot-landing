# СтройБот — лендинг

B2B-сервис для подрядчиков и поставщиков стройматериалов.
Заявка от подрядчика → конкуренция поставщиков → лучшая цена за 15 минут.

## Стек

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion (анимации)
- Lucide React (иконки)
- Plus Jakarta Sans (Google Fonts)

## Локальный запуск

```bash
npm install
npm run dev
```

Открой http://localhost:3000

## Сборка для прода

```bash
npm run build
npm run start
```

## Деплой

Подключён к Vercel. При пуше в `main` сайт автоматически пересобирается и деплоится.
